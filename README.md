# analise-preditiva-acidentes-transito-V2
Pipeline de ML para prever gravidade de acidentes de trânsito. Análise de acidentes com Random Forest, SVM e Regressão Logística. Inclui EDA, feature engineering, otimização de hiperparâmetros com RandomizedSearchCV e visualizações. Projeto acadêmico de Engenharia de Software - IA (2025).
